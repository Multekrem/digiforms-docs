# Digiforms Designer

Digiforms Designer is used to build XML-based document templates and web applications.

Designer defines:

- Data structure
- Layout and presentation
- User interaction
- Reusable components

Designer supports two primary output types:

- [PDF Documents](pdf/index.md)
- [Web Applications](web/index.md)
