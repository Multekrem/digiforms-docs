# Terminology

Understanding key terms makes it easier to navigate the documentation.

## Common Terms

**Template**  
Defines layout and structure for PDF or web output.

**XML Data Model**  
Structured representation of document or form data.

**Revision**  
A versioned state of a document template.

**Datasource**  
External or internal XML data source used within templates.

**Process Flow**  
Server-side execution logic controlling validation, integration and progression.

**Plugin**  
Custom Java-based extension executed within Digiforms Server.

For detailed technical implementation, see the relevant sections under Designer or Server.
