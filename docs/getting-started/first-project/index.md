# Your First Project

A typical Digiforms project follows these steps:

1. Define an XML data model
2. Create templates in Designer
3. Configure bindings and layout
4. Publish to Digiforms Server
5. Test and validate
6. Configure integrations if required

Projects may result in:

- A PDF document
- A web-based application
- A combined web + PDF solution
