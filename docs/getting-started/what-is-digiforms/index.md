# What is Digiforms

Digiforms is an XML-based platform for building secure digital services.

It supports:

- Structured PDF document generation
- Interactive web applications
- Integration with external systems
- Server-side process execution

Digiforms consists of two main components:

- Digiforms Designer
- Digiforms Server
